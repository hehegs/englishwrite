# englishwrite — 영어 쓰기 연습

크롬북에서 학생들이 스타일러스로 **영어 손글씨**를 연습하고 **받아쓰기**를 할 수 있는
학생용 쓰기 웹앱(PWA)입니다. 별도 설치나 스토어 없이 크롬 브라우저에서 바로 실행되고,
오프라인에서도 동작하며, "앱으로 설치"하면 전체화면 앱처럼 쓸 수 있습니다.

- **대상**: 초등 고학년 ~ 중학생
- **기술**: 빌드 도구 없는 순수 정적 PWA (HTML/CSS/JavaScript)

## 주요 기능

### 1. 자유 필기
- 영어 4선 공책 / 모눈 / 빈 종이 배경 선택
- 펜, 형광펜, 획 단위 지우개
- 스타일러스 필압(굵기 자동 변화) 반영, 손바닥 오작동 방지(펜 우선)
- 6가지 색상, 굵기 조절, 실행취소/다시실행, PNG 이미지로 저장

### 2. 따라쓰기 (Tracing)
- 4선 칸에 흐린 글씨를 깔아주면 그 위로 따라 쓰기
- 단어/뜻 표시, 발음 듣기

### 3. 받아쓰기 (Dictation)
- 단어를 소리(TTS)로 들려주고 받아쓰기 → 정답 보기로 스스로 채점
- 보통/천천히 두 가지 속도로 듣기

### 단어장
알파벳(대/소문자), 숫자, 요일·월, 학교·일상, 동사, 형용사, 짧은 문장 등
초등 고학년~중학생 수준의 카테고리별 단어/문장 제공. `js/words.js`에서 자유롭게 추가/수정.

## 실행 방법

서비스 워커(오프라인 기능) 때문에 파일을 직접 열지 말고 로컬 서버로 띄워야 합니다.

```bash
# Python 3
python3 -m http.server 8000
# 그 뒤 브라우저에서 http://localhost:8000 접속
```

### 배포 (GitHub Pages 추천)
저장소 Settings → Pages → Branch를 이 브랜치로 지정하면 정적 사이트로 바로 배포됩니다.
크롬에서 주소 표시줄의 "설치" 아이콘을 누르면 크롬북에 앱으로 설치됩니다.

## 키보드 단축키
- `Ctrl+Z` / `Ctrl+Y` : 실행취소 / 다시실행
- 연습 모드에서 `←` `→` : 이전/다음 단어, `Space` : 발음 다시 듣기

## 파일 구조
```
index.html              화면 구조
css/styles.css          스타일
js/words.js             단어/문장 데이터
js/speech.js            발음 읽어주기(Web Speech API)
js/handwriting.js       손글씨 캔버스 엔진
js/app.js               모드/도구/연습 로직 조립
manifest.webmanifest    PWA 설정
sw.js                   서비스 워커(오프라인 캐시)
icons/icon.svg          앱 아이콘
```
